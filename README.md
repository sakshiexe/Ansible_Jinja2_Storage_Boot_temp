# Ansible Storage & Boot Policy Generator

This project demonstrates infrastructure policy generation using **Ansible and Jinja2**.

## Features
- Environment-based configuration (dev/prod)
- Jinja2 templating for dynamic policies
- Ansible playbooks running on localhost
- Reusable and scalable design

## Tech Stack
- Ansible
- Jinja2
- YAML
- Git

## How to Run
```bash
ansible-playbook playbook.yml

```
# Note
Please delete existing files in the `output/` directory before running the playbook to clearly view the results of a successful execution.
